Orígenes


La historia del lenguaje de programación Python se remonta hacia finales de los 80s y principio de los 90s​, su implementación comenzó en diciembre de 1989​ cuando en Navidad Guido Van Rossum que trabajaba en el (CWI) (un centro de investigación holandés de carácter oficial que, entre otras cosas, actualmente alberga la oficina central del W3C) decidió empezar el proyecto como un pasatiempo dándole continuidad al lenguaje de programación ABC del que había formado parte del equipo de desarrollo en el CWI,​ dicho lenguaje se enfocaba en ser fácil de usar y aprender manteniendo potencia en su desempeño pero el hardware disponible en la época de su creación hacía difícil su uso y el proyecto no trascendió como se esperaba.

Van Rossum es por tanto el autor principal de Python y continúa ejerciendo un rol central decidiendo la dirección del lenguaje, en la comunidad de Python se le conoce como Benevolent Dictator for Life (BDFL). El nombre "Python" viene dado por la afición de Van Rossum al grupo Monty Python.


Características diferenciales


Python es un lenguaje de programación interpretado multiparadigma. ¿Qué quiere decir esto?.
Significa que Python no es un lenguaje compilado como lo puede ser C++ (Qué requiere de una lectura previa del documento base) sino que es uno interpretado como JavaScript (El código corre en tiempo real o al menos Python posee su consola).

Se diferencia de otros lenguajes de programación que pueden ser enfocados para un nicho y/o disciplina en especifico (Como lo es JavaScript para la Web o como lo es Java con la Programación Orientada a Objetos). Se dice que Python ocupa múltiples paradigmas ya que (Aclaro, no digo que otros lenguajes no puedan) soporta parcialmente POO, programación imperativa, y en menor medida programación funcional.

Siendo este lenguaje ideal para muchas ramas de la informática tales como: Desarrollo Backend, Scripting, Seguridad Informática, Internet of Things y donde se le puede sacar más jugo que es en Data Sciences (análisis de datos y estadísticas, redes neuronales, inteligencia artificial, etc)
